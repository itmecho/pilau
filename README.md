# Pilau

i3 ricing using [Nord Theme](https://www.nordtheme.com/) as a colour scheme

## Requirements
Arch Packages:
* `arc-icon-theme`
* `cli-visualiser`
* `compton`
* `dunst`
* `feh`
* `network-manager-applet`
* `nordic-theme-git`
* `polybar`
* `pulseaudio`
* `rofi`
* `termite`
* `ttf-font-awesome`
* `ttf-inconsolata`
* `ttf-roboto`

## Installation
```
git clone https://github.com/itmecho/pilau ~/.pilau
cd ~/.pilau
./install.sh
```
